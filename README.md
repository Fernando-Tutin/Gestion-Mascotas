# Sistema de Gestión de Mascotas

## Estudiante

## Descripción

Este proyecto desarrolla un pequeño sistema de gestión de mascotas utilizando dos enfoques de programación diferentes:

1. Programación Tradicional.
2. Programación Orientada a Objetos (POO).

Ambos programas permiten registrar y mostrar información básica de una mascota utilizando los siguientes datos:

- Nombre
- Especie
- Edad

## Reflexión

La Programación Tradicional organiza el programa mediante variables y funciones. Es adecuada para programas pequeños, pero cuando el proyecto crece puede resultar más difícil mantener y reutilizar el código.

Por otro lado, la Programación Orientada a Objetos permite representar elementos del mundo real mediante clases y objetos.
Los atributos almacenan información y los métodos definen comportamientos. 
Este enfoque facilita la organización, reutilización y mantenimiento del código, especialmente en proyectos más grandes.

En esta práctica observé que ambos programas resuelven el mismo problema, pero la Programación Orientada a Objetos ofrece una estructura más clara y escalable.
